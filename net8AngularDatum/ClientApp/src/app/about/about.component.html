Beginner-Friendly Sample Application for Angular 17 and ASP.NET Core 8.0
Build a cloud-ready application using the latest web technologies for free!
This sample application showcases a practical implementation of Angular 17 frontend and ASP.NET Core 8.0 backend, communicating through asynchronous RESTful APIs. It's designed for beginners to explore and learn from.

Key Features:
  • Modern Tech Stack: Leverages Angular 17 and ASP.NET Core 8.0 for a robust and scalable foundation.
  • Cloud-Ready: Runs seamlessly on Linux environments, including free Oracle Cloud Linux VPS hosting (free tier).
  • Open-Source: Explore the code on GitHub and adapt it for your own projects.
  • Educational Resources: Future YouTube tutorials planned to guide you through the code and concepts.

Get Started:
  1. Access the Live Demo: Experience the application in action at https://datum.vagweb.com.
  2. Download the Code: Clone the repository from GitHub Datum calculation.
  3. Explore and Learn: Dive into the code and experiment with its features.
  4. Build Your Own: Use it as a foundation for your own web development projects.

Important Note:
The HKG80 to WGS84 calculation logic is based on a 20-year-old model and may not provide high precision. Its purpose is primarily for demonstrating Angular and ASP.NET Core concepts.
